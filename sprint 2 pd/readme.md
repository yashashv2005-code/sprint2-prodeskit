# Personal Finance Dashboard

A simple and responsive Personal Finance Dashboard built using HTML, CSS, and Vanilla JavaScript. This project helps users track their daily expenses, monitor their spending habits, and visualize their expenses through interactive charts.

## Project Overview

Managing personal finances can be challenging without proper tracking. This dashboard provides an easy way to record expenses, view spending summaries, and analyze expenses by category.

The project was developed as part of a frontend development sprint with a focus on DOM manipulation, local storage, data visualization, and responsive design.

## Features

### Expense Management
- Add new expenses
- Edit existing expenses
- Delete expenses
- View all transactions in a structured table

### Dashboard Summary
- Total Expenses
- Remaining Balance
- Total Number of Transactions
- Current Date Display

### Data Persistence
- Uses Local Storage to save data
- Expenses remain available even after refreshing the browser

### Data Visualization
- Interactive Doughnut/Pie Chart using Chart.js
- Category-wise expense breakdown

### Additional Features
- Currency Selector (INR, USD, EUR, GBP)
- Low Balance Warning
- PDF Report Download
- Basic Spending Insights

1. Download or clone the repository.

```bash
git clone <repository-url>
```

2. Open the project folder.

3. Launch `index.html` in your browser.

No additional installation or setup is required.

## How It Works

1. Enter expense details in the form.
2. Submit the form to add an expense.
3. The expense will appear in the transactions table.
4. Dashboard statistics update automatically.
5. Data is stored in Local Storage.
6. Charts and insights refresh whenever data changes.
